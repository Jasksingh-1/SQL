# SQL for Data Analytics

This repository contains structured SQL queries covering
fundamental to advanced SQL concepts with a strong analytics focus.

## Topics Covered
- SQL fundamentals & query execution order
- Core querying (SELECT, WHERE, GROUP BY, HAVING)
- Joins (Inner, Left, Right, Full, Anti joins)
- Set operators (UNION, INTERSECT, EXCEPT)
- String, number, and date functions
- DDL & DML commands

## Approach
Each SQL file includes:
- Business-oriented problem statements
- Clean, readable SQL queries
- Practical analytics use cases

## Databases
- PostgreSQL / SQL Server compatible syntax
