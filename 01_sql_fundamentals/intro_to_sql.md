# Introduction to SQL

This section covers the foundational concepts required to understand
how SQL works with databases and why it is essential for data analytics.

---

## What is SQL?
SQL (Structured Query Language) is a standard language used to communicate
with relational databases. It allows users to retrieve, insert, update,
delete, and manage data efficiently.

---

## What is a Database?
A database is an organized collection of structured data stored electronically.
Databases allow data to be stored, accessed, and managed efficiently.

---

## What is DBMS?
DBMS (Database Management System) is software that manages databases and
controls access to data.

Examples:
- PostgreSQL
- MySQL
- SQL Server
- Oracle

---

## DBMS vs SQL Server
- DBMS is a general concept that refers to database management software.
- SQL Server is a specific DBMS developed by Microsoft.

---

## Types of Databases
- Relational Databases (PostgreSQL, MySQL, SQL Server)
- NoSQL Databases (MongoDB, Cassandra)
- Data Warehouses (Snowflake, Redshift)
- Cloud Databases

---

## Types of SQL Commands
- DDL (CREATE, ALTER, DROP)
- DML (INSERT, UPDATE, DELETE)
- DQL (SELECT)
- DCL (GRANT, REVOKE)
- TCL (COMMIT, ROLLBACK)

---

## Why Learn SQL?
- SQL is the foundation of data analytics
- Used across BI tools, Python, and cloud platforms
- Enables efficient data analysis and reporting
- Essential for roles like Data Analyst, BI Analyst, and Data Scientist

---

## Tools & Environment
SQL can be executed using:
- PostgreSQL
- SQL Server (SSMS)
- MySQL
- Cloud platforms like Snowflake

This repository focuses on analytics-oriented SQL use cases.
